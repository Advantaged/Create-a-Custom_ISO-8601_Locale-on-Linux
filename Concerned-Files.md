# Concerned-Files-.md

All listed files below allow every user of the machine to get homogeneous locale.
* ***NOTE:***
 * you should know what you are doing! Don't play with or touch these files if you don't know the resulting effect.
 * In case of wrong handling of these files, you will not more able to install any package-program on your machine.
 * Please, be very careful!
```bash
~/.bashrc

~/.config/locale.conf

~/.config/plasma-localerc

/etc/bash.bashrc

/etc/profile.d/locale.sh      ### no need to edit!

/etc/locale.conf

/etc/profile                  ### no need to edit!

/etc/skel/.config/locale.conf
```
